# 论文提交补充材料文件清单

本文件夹用于投稿时单独提交 Supplementary Information 和相关源数据。

## 建议上传文件

- `ESIN_supplementary_information.pdf`：补充材料主文件，包含 Supplementary Figs. S1-S3 和 Supplementary Tables S1-S2。
- `ESIN_supplementary_information.tex`：补充材料 LaTeX 源文件。
- `ESIN_Supplementary_Figure_S1.pdf`：MAUP and shifted-origin sensitivity。
- `ESIN_Supplementary_Figure_S2.pdf`：Placebo randomization checks。
- `ESIN_Supplementary_Figure_S3.pdf`：Transfer, leave-one-out and external sign-control experiments。
