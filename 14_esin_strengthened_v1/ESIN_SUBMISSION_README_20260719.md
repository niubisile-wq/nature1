# ESIN submission README

Date: 19 July 2026

This folder contains a strengthened no-EGMS-credential submission package for *Earth Science Informatics*.

## Primary upload files

- `ESIN_strengthened_v1.pdf`: compiled manuscript for review.
- `ESIN_strengthened_v1.tex`: LaTeX source.
- `esin_references.bib`: bibliography.
- `sn-jnl.cls` and `sn-mathphys-ay.bst`: Springer template/style files used for local compilation.
- `ESIN_Figure_0.pdf` to `ESIN_Figure_9.pdf`: figure files for LaTeX compilation.
- `ESIN_Figure_0.tiff` to `ESIN_Figure_9.tiff`: high-resolution figure files if separate upload is requested.
- `ESIN_Figure_*_source_data.csv`: figure source-data tables.
- `ESIN_COVER_LETTER_20260719.md`: editable cover letter.
- `ESIN_HIGHLIGHTS_20260719.md`: optional highlights.
- `ESIN_DECLARATIONS_CHECKLIST_20260719.md`: declarations and upload checklist.

## Evidence and audit files

- `NO_EGMS_CREDENTIAL_SUBMISSION_ROUTE_20260719.md`: final validation framing for the no-credential route.
- `ESIN_SUBMISSION_GATE_STATUS_20260719.json` and `.md`: completed gates and future EGMS hooks.
- `regional_non_egms_strengthening_v1.csv` and `.md`: non-EGMS regional transfer evidence.
- `external_control_synthesis_v1.csv`: DWR/TRE and Cyprus EGMS external-control summary.
- `source_data_inventory_strengthened_v1.csv`: file inventory and SHA256 hashes.

## Boundary

The manuscript does not claim completed Po/Rhone EGMS strong-subsidence validation. Those scripts and folders remain in the package as future upgrade hooks only.
