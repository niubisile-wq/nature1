# ESIN Strengthened Manuscript Package

This folder contains the public, reproducibility-focused package for the Earth Science Informatics manuscript.

## Manuscript Files

- `ESIN_strengthened_v1.tex`
- `ESIN_strengthened_v1.pdf`
- `ESIN_supplementary_information.tex`
- `ESIN_supplementary_information.pdf`
- `esin_references.bib`
- `ESIN_strengthened_v1.bbl`
- `sn-jnl.cls`
- `sn-mathphys-ay.bst`

## Figures and Source Data

Final figure PDFs/SVGs and corresponding `*_source_data.csv` files are included for manuscript figures and supplementary figures. The source-data inventory is `source_data_inventory_strengthened_v1.csv`.

## Reproducibility Scripts

The Python scripts in this folder regenerate the submitted figures and the lightweight audit tables used in the manuscript. Large third-party remote-sensing products are not mirrored in this repository; public sources and access routes are documented in the manuscript and data-availability statement.
