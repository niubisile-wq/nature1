# Cross-Paper Comparison v1
Date: 2026-07-13

## Scope

Compare the Nature observability-bias package against five existing peer packages in the workspace:

- `Second Batch 1`
- `Second Batch 2`
- `Third Batch 1`
- `power_se`
- `battery_soh`

This is a comparison-only note. It does not modify those packages.

## Comparison rubric

The comparison uses two axes:

1. Scientific depth: problem breadth, evidence chain strength, validation breadth, and claim discipline.
2. Submission maturity: package completeness, figure/table handoff, boundary control, and final release readiness.

## What Nature already does better

1. The core scientific story is broader and more distinctive than the benchmark-style packages.
2. The evidence chain is multi-region and explicitly transfer-aware.
3. The claim boundary is tighter than the more generic benchmark packages because EGMS is kept as an explicit upgrade path.
4. The package already has the right shape for reviewer defense: claim map, risk audit, consistency audit, release readiness, and transfer audit.

## What Nature still needs to beat them on the full scoreboard

1. A true raw-input rerun path for the raster-backed Chao Phraya steps.
2. A minted DOI and repository URL with placeholders removed.
3. A fully closed all-region cell-level hierarchical fit, or a formal no-fit note that is explicit enough to survive review.
4. A submission-package polish layer comparable to the best final-note packages in `power_se` and `battery_soh`.
5. A clean cross-paper comparison appendix so the Nature package can be defended as the strongest overall package, not only the strongest science story.

## Bottom line

Nature is already the strongest science story in this set, but it is not yet the cleanest submission package. The remaining work to "beat the five" is not more narrative polish first; it is closing the reproducibility and release gaps so the scientific advantage can be carried all the way through submission readiness.
